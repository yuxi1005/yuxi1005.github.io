Record of Working on LS

2025.03.03

  1. 修改实体命名以符合javabean命名规范
  2. 使用Lombo来减少不必要的getter和setter代码量
  3. 登陆localhost8080需要登陆是因为Pom.xml中有spring-boot-starter-test依赖。这意味着 Spring Security 已经被集成到你的项目中，即使没有显式配置它，Spring Boot 也会启用默认的安全配置。
  4. 访问localhost出现404的原因：getMapping参数中应该用实体的属性名字而不是随便起的id，比如：@**GetMapping**("/{courseId}/notes")
